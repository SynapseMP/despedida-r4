# Despedida R4 · Radiodiagnóstico

Invitación web para la fiesta de despedida de los R4 de Radiodiagnóstico del Hospital Universitario de Salamanca.

**15 de mayo de 2026 · 20:00h · Hotel Alameda Palace · Salamanca**

## Contenido del repo

- `index.html` — la web completa (HTML + CSS + JS en un único archivo)
- `fotos/` — retratos de los R4 + 15 imágenes del collage

## Despliegue

Servida desde GitHub Pages (rama `main`, carpeta raíz). Cualquier cambio mergeado en `main` se publica en 30-60 segundos.

## Backend

La votación de menú, libro de firmas, alergias y confirmación de asistencia se persisten en una hoja Google a través de un Apps Script. La URL del script vive embebida en `index.html` (`SCRIPT_URL`).
